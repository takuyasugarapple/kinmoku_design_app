Bootstrap

Bootstrap を使ってみましょう。 Bootstrap （ブートストラップ）とは、 Twitter社が開発した、Webサイトやアプリケーションを構築するためのHTML/CSS/JavaScriptの フレームワーク のことです。( https://getbootstrap.com/ )

Previous Next

自分で0から創ることも重要ですが bootstrap を利用するとレスポンシブも、簡単にデザイン性が高くなるため依存するまではいかずとも 柔軟に利用 することは重要です。

Previous Next

このページ[ https://getbootstrap.com/ ]を開き、「 BootstrapCDN 」の欄にある「 CSS only 」のスニペットをコピーし、index.htmlのheadタグの中にペーストします。

Previous Next

さらに今回は使用しませんが「 JS, Popper.js, and jQuery 」のスニペットもコピーし、bodyタグにペーストしていきましょう。これから書く内容はこのjsのコードの上に書いていきます。

Previous Next

基本的に bootstrap のcssやjavascriptファイルは触りません。もしもその部分をいじりたいときはその bootstrap のクラスの後に任意のクラスもしくはidを加え、そのcssをいじります。
